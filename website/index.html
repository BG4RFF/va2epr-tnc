Introduction

Greetings and Salutations! My name is Thomas, and I'd like to welcome you to my little pit stop on the information superhighway. I live in Gatineau, Quebec, Canada with my wife Bridget. I am currently a student in the Electronics Technology program at Heritage College. I'm in the final year of my studies, and I created this website to showcase my final year project. The project incorporates one of my favourite hobbies, Amateur Radio. The goal is to design and build a Terminal Node Controller (i.e. a radio modem). Additionally, I'm going to be making a portable version with an integrated GPS receiver. The target application of the portable version is APRS tracking. If you would like to contact me, my e-mail address is linuxgeek@gmail.com.

Current Status

I've created an initial schematic, a bill of materials, and I've ordered parts. I've also tested portions of the circuits on breadboard and written some proof of concept code for the firmware. I'm just beginning to work on this website and push things to github. A formal project plan is being drafted, and work is beginning on the printed circuit board layout. I have a lot to add to the site, so check back for updates.
